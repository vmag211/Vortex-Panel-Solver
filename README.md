# Vortex Panel Solver: 2D Aerodynamics Engine

This project is a custom-built 2D aerodynamics engine written in Python. It calculates the airflow, air pressure, and lift generated by wing shapes (airfoils) using a technique called a **panel method**. 

Instead of using heavy, slow simulation software, this program breaks the outline of a wing down into tiny straight lines (panels). It then uses linear algebra to calculate how the air interacts with every single panel. 

## Features
* **Custom Wing Generation:** Automatically builds the exact geometry for any standard NACA 4-digit airfoil (like the NACA 2412).
* **Pressure Mapping:** Calculates the Pressure Coefficient (Cp) at every point on the wing to find exactly where suction and lift are happening. 
* **The Kutta Condition:** Includes advanced mathematical rules to ensure the simulated air flows smoothly off the sharp trailing edge (back edge) of the wing, just like it does in real life.

## How to Install
To run this code on your own machine, you will need Python 3 installed. 

1. Clone this repository to your computer.
2. Create a virtual environment (optional but recommended).
3. Install the required engineering math libraries by running:
   `pip install -r requirements.txt`

## How to Run the Code
There are three main programs you can run in your terminal:

**1. The Math Test:** `python test_cylinder.py`
This tests the core math engine against a perfect circle. Since a perfect circle has an exact, proven mathematical answer, this proves the custom physics engine is calculating correctly without errors.

**2. The Physics Test:** `python test_airfoil.py`
This runs the engine on a perfectly symmetrical wing (NACA 0012). It verifies that when the wind hits a flat wing straight on, it correctly produces exactly zero lift.

**3. The Visual Demo:** `python run_demo.py`
This is the main event. It takes a curved wing (NACA 2412), runs the aerodynamics simulation, and automatically generates three visual graphs in the `results/` folder:
* The physical shape of the wing.
* A graph showing the air pressure (suction) lifting the wing.
* A graph comparing the wing's lift to textbook aerodynamic theory.
